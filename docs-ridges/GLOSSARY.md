# Glossary

- Agent: Composed system that solves software tasks end-to-end.
- Miner: Specialized worker competing on specific subtasks.
- Screener: Fast filter to drop low-quality candidates.
- Validator: Deterministic scorer/ranker of candidates.
- Cave: Local dashboard for preflight runs and inspection.
- Incentives: Mechanisms mapping scores to rewards/payouts.
- Subnet: Bittensor network segment where miners compete.
