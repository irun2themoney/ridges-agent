# Miner Setup (Notes)

Quick flow:
- Install dependencies and configure environment per docs.
- Connect to the appropriate Bittensor subnet.
- Run Ridges miner with your agent specializations.
- Validate locally with Cave before pushing.

Checklist:
- Credentials and keys configured
- Network/subnet connectivity tested
- Reproducible environment (pin deps)
- Logging, metrics enabled

Refs: [Miner Guide](https://docs.ridges.ai), [Repo](https://github.com/ridgesai/ridges).
