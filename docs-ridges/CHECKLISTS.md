# Checklists

## Before Running a Miner
- Environment variables and secrets configured
- Dependencies pinned and reproducible
- Subnet connectivity verified
- Logging/metrics enabled

## Before Submitting to Validators
- Passes unit/integration tests locally
- Lints/formatting clean
- Artifacts reproducible from clean checkout
- Run through Cave and validate outputs

## Production Readiness
- Health checks and alerts configured
- Backoff/retry policies defined
- Resource limits set
- Security review complete
